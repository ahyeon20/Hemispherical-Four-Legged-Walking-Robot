# Hemispherical-Four-Legged-Walking-Robot
This is a hemispherical quadruped robot. It implements inverse kinematics and allows communication between modules.

<img width="348" height="267" alt="image" src="https://github.com/user-attachments/assets/fadf63c0-a944-4e2a-9255-eb7af164ae39" />

**역기구학 기반 보행 (Inverse Kinematics based Gait):** 다리 움직임을 정밀하게 제어하여 다양한 지형에서 안정적인 보행이 가능

**하드웨어 (Hardware):**

  외피 재질: PLA  
  구동모터:  
    -2XL430-W250-T X4  
    -CLS400MD  
  MCU: 아두이노 메가  
  모터드라이버: 다이나믹셀 쉴드
  
**소프트웨어 (Software):**

  Arduino IDE  
  C++  
  보행방식: creep gait
